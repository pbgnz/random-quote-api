Random Quote API

Get random quotes from Goodreads with a simple REST API

📚 API Documentation 🔗 GitHub

Quick Start

Fetch a random quote:

GET /api/v1/quotes/random

Get multiple quotes with an optional count parameter:

GET /api/v1/quotes?count=5

View full API documentation at /api/docs
